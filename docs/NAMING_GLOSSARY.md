# Naming Glossary — TokyOS & Tokyo IA

## Termos Oficiais

| Termo | Significado | Uso |
|-------|-------------|-----|
| **TokyOS** | Sistema operacional + plataforma | Nome do produto, interface, serviços |
| **Tokyo IA** | Inteligência artificial do sistema | Agentes, memória, automação, decisões |
| **TokyOS Portal** | Dashboard principal | Tela inicial após login |
| **TokyOS Desktop** | Ambiente de trabalho | Launcher, dock, widgets, wallpaper |
| **TokyOS Store** | Loja de aplicativos | Catálogo e instalação de apps |
| **TokyOS Settings** | Configurações do sistema | Personalização, contas, ambiente |
| **TokyOS Logs** | Central de logs | Visualização de logs do sistema |
| **TokyOS Backup** | Gerenciamento de backups | Snapshots, restic, timeshift |
| **TokyOS Services** | Gerenciamento de serviços | Systemd service manager |
| **Tokyo IA Agent** | Agente de IA | Hermes, Paperclip, Automação RPA |
| **Tokyo IA Brain** | Cérebro da IA | Memória neural, rag, embeddings |
| **Tokyo Memory** | Memória persistente | Vetores, histórico, contexto |
| **Tokyo Voice** | Assistente de voz | STT, TTS, Jarvis |
| **Tokyo Vision** | Visão computacional | OCR, análise de imagem |
| **Tokyo Workflows** | Motor de workflows | Automação de processos |
| **Tokyo Approval Guard** | Guardião de aprovações | Auditoria, compliance, approval flow |
| **Tokyo Policy Engine** | Motor de políticas | Regras de negócio, permissões |

## Regras de Uso

1. **TokyOS** sempre com "T" maiúsculo e "OS" maiúsculo
2. **Tokyo IA** com "IA" maiúsculo (não "Ia" ou "ai")
3. Em textos longos, usar "TokyOS" e "Tokyo IA" por extenso na primeira menção; depois "plataforma" ou "sistema" são aceitáveis
4. Nunca usar "TokiOS" (grafia antiga, incorreta)
5. Nunca usar "ClawOS" ou "KiraCore" em materiais de marketing ou interface
